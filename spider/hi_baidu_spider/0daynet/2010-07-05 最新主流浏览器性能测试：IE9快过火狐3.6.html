最新主流浏览器性能测试：IE9快过火狐3.6

据国外媒体报道，上周，挪威浏览器厂商Opera发布了新版浏览器Opera 10.6，为用户带来了一系列的性能和功能改进。与此同时，Opera还宣称，Opera 10.6是当今全球运行最为快速的一款浏览器。

众所周知，全球流行的浏览器有5款，分别为谷歌Chrome、Opera、Mozilla Firefox、微软IE和苹果Safari等。

下面，笔者将使用以下3个流行的互联网测试套件针对上述5个主流浏览器进行测试，分别为：Sunspider Javascript基准套件、Celtic Kane的JSBenchmark Javascript基准套件、Futuremark的Peacekeeper浏览器基准套件。

参与测试的浏览器版本分别有：Opera 10.5、Opera 10.6 、Chrome 6.0.408.1、Chrome 5.0.375.86、FireFox 3.6.4 、Safari 5.0 (7553.16) 、IE8、IE9的第三个开发者预览版。

以下是测试结果：

一、SunSpider基准测试结果：

1. Opera 10.5 353.4ms +/- 1.1%

2. Chrome 6.0.408.1 489.6ms +/- 3.9%

3. Opera 10.6 517.4ms +/- 5.7%

4. Safari 5.0 (7553.16) 600.4ms +/- 1.1%

5. Chrome 5.0.375.86 635.0ms +/- 3.6%

6. IE9第三个平台预览版 807.4ms +/- 12.1%

7. FireFox 3.6.4 1396.6ms +/- 14.6%

8. Internet Explorer 8 7228.8ms +/- 9.7%

二、Celtic Kane 的JSBenchmark 基准测试结果：

1. Chrome 5 459 ± 0

2. Opera 10.6 387 ± 0

3. Chrome 6.0.408.1 355 ± 0

4. Safari 5.0 (7553.16) 252 ± 0

5. Opera 10.5 211 ± 0

6. IE9第三个平台预览版 177 ± 0

7. FireFox 3.6.4 100 ± 0

8. Internet Explorer 8 59 ± 15

三、Futuremark Peacekeeper基准测试结果：

1. Opera 10.6 5244 点

2. Chrome 6.0.408.1 5162点

3. Chrome 5.0.375.86 4897点

4. Opera 10.5 3323点

5. Safari 5.0 (7553.16) 2606点

6. Firefox 3.6.4 1939点

7. Internet Explorer 9 1919点

8. Internet Explorer 8 829点

如上所示，IE9第三个开发者预览版的运行速度已经超过了Firefox 3.6，这对于微软来讲是一个很不错的好消息。在Sunspider Javascript基准测试中，IE9的运行速度已经基本接近IE8的10倍。而Opera 10.6则并没有像Opera公司声称的那样成为全球最快速的浏览器，反而是Opera 10.5仍然远远领先于其他的竞争对手。

与此同时，谷歌Chrome 6经常会出现崩溃的现象，而Chrome 5虽然相对稳定，但相较于Firefox 3.6和Opera 10.6仍然有所不足。

笔者认为，Opera 10.6中最大的改进并不是其速度，而是绑定了免费的杀毒软件AVG。
